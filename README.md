# TP2-Mundial

# Seguí estos pasos para levantar el proyecto

### 1. Requisitos Previos
# Python 3.10 o superior
# MySQL Server
# Postman (o similar) para probar los endpoints

### 2. Base de Datos
# 1. Abrí el gestor de MySQL (DataGrip, Workbench, etc)
# 2. Ejecutá el script de creación de tablas que se encuentra en:  
   `backend/database/scripts/database_tpbackend.sql`

### 3. Entorno Virtual
# Desde la terminal, parado en la carpeta raíz del proyecto (`TP2-MUNDIAL`), ejecutá:

# 1. Crear el entorno virtual
# python -m venv .venv

# 2. Activar el entorno (Windows PowerShell)
# .\.venv\Scripts\Activate.ps1

# 3. Instalar todas las dependencias
# pip install -r requirements.txt
